# Stack

A secure, expandable, array-based stack of strings implemented in C, C++, and Java.

## Building and Running Tests

### C

```
cd c
gcc -o test_stack test_stack.c stack.c
./test_stack
```

### C++

```
cd cpp
g++ -std=c++17 -o test_stack test_stack.cpp Stack.cpp
./test_stack
```

### Java

```
cd java
javac Stack.java StackTest.java
java StackTest
```
